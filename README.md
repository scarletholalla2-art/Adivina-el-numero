Nombre del proyecto:
Sistema “Adivina el número”.

Integrantes:
Scarleth Angelina Olalla Ojeda

Objetivo del sistema:
Desarrollar un sistema interactivo utilizando Python que permita a la computadora adivinar el número pensado por el usuario mediante intentos y respuestas proporcionadas durante el juego. El objetivo es aplicar conocimientos relacionados con diseño, programación y organización de software, integrando los contenidos aprendidos durante la asignatura.

Descripción de funcionalidades:
El sistema cuenta con las siguientes funcionalidades:

Iniciar el juego.
Permitir que el usuario piense un número.
Generar intentos automáticos para adivinar el número.
Recibir respuestas del usuario (mayor, menor o correcto).
Continuar el proceso hasta encontrar el número correcto.
Mostrar el resultado final y la cantidad de intentos realizados.
